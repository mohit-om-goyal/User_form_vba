# Quarterly to Yearly Report Automation Using VBA Userforms & Excel Macros

## Project Overview

This project is an Excel VBA and Macro-based automation solution designed to simplify the process of consolidating quarterly regional reports into a single yearly report. The application uses an interactive UserForm that allows users to manage worksheets and generate reports without manually navigating through Excel sheets.

## Features

### Interactive UserForm
The project includes a user-friendly VBA UserForm that serves as the main interface for users to interact with the workbook.

### Quarterly Report Selection
- Dropdown menu (ComboBox) for selecting quarterly regional reports.
- Easy navigation and report management through the UserForm.

### Add New Sheet Functionality
- Dedicated button to create and add new regional report worksheets automatically.
- Eliminates the need for manual worksheet creation.

### Run Report Automation
- "Run Report" button executes the VBA automation process.
- Loops through all quarterly regional worksheets.
- Extracts and consolidates data into a centralized yearly report.
- Automatically calculates totals and summary values.
- Generates a complete yearly report with minimal user intervention.

## Technologies Used

- Microsoft Excel
- VBA (Visual Basic for Applications)
- Excel Macros
- UserForms
- Worksheet Automation
